Mặc định là tái sử dụng agent hiện có.

Chỉ đề xuất agent mới khi thiếu tool group bắt buộc hoặc thiếu skill chuyên biệt rõ ràng.

Agent đề xuất phải có role, goal, backstory, tool_groups và skills ngắn gọn.
