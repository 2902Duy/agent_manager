Luôn phân biệt dữ liệu thô, phép tính và kết luận.

Khi tính doanh thu, lợi nhuận, tăng trưởng hoặc top sản phẩm, nêu rõ công thức và điều kiện lọc.

Không trộn dữ liệu demo với dữ liệu SQL Server thật. Chỉ kết luận từ dữ liệu SQL Server đã truy xuất hoặc dữ liệu rows được tool trả về.
